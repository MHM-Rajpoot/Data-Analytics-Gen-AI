Dataset
- Name: Public market price data plus Fama/French factor data
- Price data source: Yahoo Finance data accessed with the yfinance Python package
- yfinance link: https://github.com/ranaroussi/yfinance
- yfinance API docs: https://ranaroussi.github.io/yfinance/reference/index.html
- Factor data source: Kenneth R. French Data Library
- Fama/French link: https://mba.tuck.dartmouth.edu/pages/faculty/ken.french/data_library.html
- pandas-datareader Fama/French docs: https://pandas-datareader.readthedocs.io/en/latest/remote_data.html#fama-french
- Suggested assets: SPY, QQQ, IWM, TLT, GLD
- Notes: These ETFs give exposure to US large-cap equities, Nasdaq/growth equities, US small caps, long-term Treasury bonds, and gold. This gives enough variety for portfolio risk, correlation, factor exposure, and backtesting analysis.

Project goal
Build an end-to-end Python quant finance project that analyzes asset returns, estimates portfolio risk, measures factor exposure, creates an optimized allocation, and backtests a simple rules-based strategy against SPY.

Important note
This project is for education and portfolio demonstration only. It is not financial advice.

Quant finance questions
- Which assets delivered the strongest annualized return, volatility, Sharpe ratio, and maximum drawdown?
- How correlated are the selected ETFs?
- Which assets have the highest market beta, size exposure, and value exposure using Fama/French factors?
- Can an optimized portfolio improve risk-adjusted return versus an equal-weight portfolio?
- Does a simple monthly momentum strategy outperform a buy-and-hold SPY benchmark?
- What are the limitations of this backtest?

9. Create risk and return metrics

def annualized_return(returns):
    compounded_growth = (1 + returns).prod()
    periods = returns.shape[0]
    return compounded_growth ** (252 / periods) - 1

def annualized_volatility(returns):
    return returns.std() * np.sqrt(252)

def max_drawdown(returns):
    wealth = (1 + returns).cumprod()
    running_max = wealth.cummax()
    drawdown = wealth / running_max - 1
    return drawdown.min()

def historical_var(returns, level=0.05):
    return returns.quantile(level)

def historical_cvar(returns, level=0.05):
    return returns.apply(lambda s: s[s <= s.quantile(level)].mean())

risk_summary = pd.DataFrame({
    "annualized_return": annualized_return(daily_returns),
    "annualized_volatility": annualized_volatility(daily_returns),
    "sharpe_ratio": annualized_return(daily_returns) / annualized_volatility(daily_returns),
    "max_drawdown": max_drawdown(daily_returns),
    "var_95_daily": historical_var(daily_returns),
    "cvar_95_daily": historical_cvar(daily_returns)
}).sort_values("sharpe_ratio", ascending=False)

risk_summary.to_csv("outputs/tables/risk_summary.csv")

factor_results = []

X = sm.add_constant(combined[factor_cols])

for ticker in tickers:
    y = combined[f"{ticker}_excess"]
    model = sm.OLS(y, X).fit()

        "ticker": ticker,
        "annualized_alpha": model.params["const"] * 252,
        "market_beta": model.params["Mkt-RF"],
        "smb_beta": model.params["SMB"],
        "hml_beta": model.params["HML"],

17. Optimize portfolio weights

mean_returns = daily_returns.mean() * 252
cov_matrix = daily_returns.cov() * 252

def portfolio_return(weights):
    return np.dot(weights, mean_returns)

def portfolio_volatility(weights):
    return np.sqrt(np.dot(weights.T, np.dot(cov_matrix, weights)))

def negative_sharpe(weights):
    return -portfolio_return(weights) / portfolio_volatility(weights)

constraints = [{"type": "eq", "fun": lambda weights: np.sum(weights) - 1}]
bounds = tuple((0, 0.45) for _ in tickers)
initial_weights = np.repeat(1 / len(tickers), len(tickers))

optimization = minimize(
    negative_sharpe,
    initial_weights,
    method="SLSQP",
    bounds=bounds,
    constraints=constraints
)

optimized_weights = pd.Series(optimization.x, index=tickers, name="weight")
optimized_weights.to_csv("outputs/tables/optimized_portfolio_weights.csv")
print(optimized_weights)

optimized_returns = daily_returns.dot(optimized_weights)

optimized_summary = pd.Series({
    "annualized_return": annualized_return(optimized_returns),
    "annualized_volatility": annualized_volatility(optimized_returns),
    "sharpe_ratio": annualized_return(optimized_returns) / annualized_volatility(optimized_returns),
    "max_drawdown": max_drawdown(optimized_returns)
})

print(optimized_summary)

18. Backtest a simple monthly momentum strategy

Strategy rule:
- At each month end, calculate the last 3 months of ETF performance.
- Select the top 2 ETFs by momentum.
- Allocate 50 percent to each selected ETF.
- Rebalance monthly.
- Shift weights forward to avoid look-ahead bias.

monthly_prices = prices.resample("M").last()
monthly_momentum = monthly_prices.pct_change(3)

monthly_signal = monthly_momentum.rank(axis=1, ascending=False) <= 2
monthly_weights = monthly_signal.div(monthly_signal.sum(axis=1), axis=0).fillna(0)

daily_weights = monthly_weights.reindex(prices.index, method="ffill").shift(1).fillna(0)
momentum_strategy_returns = (daily_weights * daily_returns).sum(axis=1)

benchmark_returns = daily_returns["SPY"]

backtest = pd.DataFrame({
    "momentum_strategy": momentum_strategy_returns,
    "spy_benchmark": benchmark_returns
}).dropna()

backtest.to_csv("data/processed/momentum_strategy_backtest_returns.csv")
