## Structure

Every topic file follows this order:

1. Definition-Based Concept MCQs
2. Beginner
3. Intermediate
4. Advanced

Each difficulty level contains a Core MCQs section followed by a Scenario-Based Questions section. Every question includes four answer options, a clearly marked correct answer, a short explanation, and one of the required tags.
